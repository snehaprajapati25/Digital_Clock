# Digital_Clock

A simple and responsive Digital Clock built using HTML5, CSS3, and JavaScript. The application displays the current local time and date in real-time and updates automatically every second.

## Features
Real-time clock updates every second
Displays current local date and time
Clean and modern user interface
Responsive design
Built using vanilla JavaScript


## Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
